Description MOFgeoDB

General Setting The MOFgeoDB is a SQlite database using the spatial extension spatialite and serves our data of vegetation (mainly trees) and study plots for ecosystem research (forest structure, experiments) over time. Additional there are some auxiliary datasets (geopackage) of infrastructure for visual reasons. QGIS is used as back-end for analysis and visualization.
